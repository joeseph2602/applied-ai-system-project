# Reflection and Ethics

**Limitations and Biases:**
- The system relies on user input and simple rules, so it may not handle complex or ambiguous pet care scenarios.
- There is no built-in understanding of pet health beyond basic status labels.

**Potential Misuse and Prevention:**
- The system could be misused if someone relies on it for critical pet health decisions. To prevent this, always recommend consulting a veterinarian for serious issues.
- Logging helps track actions and can be reviewed to detect misuse.

**Surprises During Reliability Testing:**
- It was surprising how easily automated tests caught scheduling conflicts and confirmed correct task completion. Logging made it easy to trace and verify system behavior.

**Collaboration with AI:**
- The AI assistant helped by suggesting a clear logging and reliability testing structure, which improved the project.
- One flawed suggestion was to overcomplicate the architecture, but keeping things simple worked better for this project.
